A simple finance tracker where you can add, update, and delete transaction, as well as be able to view them in table.

**Techstack**
Backend: Springboot, Spring Data JPA, Java
Frontend: Python, Streamlit
Database: Postgres SQL

**How to use**
Select an option: Create new, update, or delete
- Fill in the blank values to add or update transaction.
- Enter id value to update or delete transaction.
